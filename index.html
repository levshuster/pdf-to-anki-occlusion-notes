How to Import Saved .TXT File Into Anki

  1. Navigate to the destination deck
  2. Select "File"-->"Import" (or press Ctrl+Shit+I)
  3. Leaving the default file type (Text Seperated by Tabs or Semicolons) select the .TXT file
  4. Click "Open"
  5. Ensure "Type" is set to "Cloze"
  6. Click "Feild seperated by" and enter "\t"
  7. Click "Import"
